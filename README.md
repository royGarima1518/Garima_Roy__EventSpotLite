# EventSpotLite

Welcome to **EventSpotLite**, a web application designed to help users discover and plan amazing events in their area with ease and elegance.

## Features

- **Responsive Navigation Header**: A sticky header with navigation links and mobile menu support.
- **Hero Section**: Welcoming users with a brief description and action buttons.
- **Image Slider**: Two image sliders showcasing events or related images with smooth transitions.
- **Interactive Envelope Popup**: Clickable envelope that reveals a modal with a message.

## Technologies Used

- **React**: JavaScript library for building user interfaces.
- **Lucide-React**: Icons for various UI elements.
- **CSS**: Styles for layout and design.

## Installation

To run the project locally, follow these steps:

1. Clone the repository:
   ```bash
   https://github.com/royGarima1518/Garima_Roy__EventSpotLite.git
2. Navigate into the project directory:
   cd my-app
3. Install the dependencies:
   npm install
4. Start the development server:
   npm start
5. Open your browser and visit http://localhost:3000 to view the application.

## Usage
- **Navigation:** Use the navigation links to explore different sections of the app.
- **Action Buttons:** Click on the action buttons in the hero section to trigger alerts.
- **Image Sliders:** Browse through images using the sliders.
- **Envelope Popup:** Click the envelope to display a message modal.
